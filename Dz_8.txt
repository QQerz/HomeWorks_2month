CREATE - команда для создания новой таблицы данных

INSERT - команда для добавления данных в таблицу

SELECT - команда для получения данных из выбранной таблицы. Следующей командой можно вывести все данные из таблицы:
															ВЫБЕРИТЕ * ИЗ <имя_таблицы>
UPDATE - команда для обновления таблицы данных.

DROP - команда для удаления строк из таблицы, полной таблицы инфекции или полной базы данных.
